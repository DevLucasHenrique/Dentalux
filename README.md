## ⚙️ Como rodar localmente
```bash
# Clone o repositório
git clone https://github.com/DevLucasHenrique/dentalux.git

# Entre na pasta
cd dentalux

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
